# Computer Networks Learning Path

## Topics to Study

1. **Introduction to Networks**
   - What is a Network
   - Types of Networks (LAN, MAN, WAN)
   - Network Topology

2. **OSI Model**
   - Overview of 7 Layers
   - Physical Layer (Layer 1)
   - Data Link Layer (Layer 2)
   - Network Layer (Layer 3)
   - Transport Layer (Layer 4)
   - Session Layer (Layer 5)
   - Presentation Layer (Layer 6)
   - Application Layer (Layer 7)

3. **TCP/IP Model**
   - Link Layer
   - Internet Layer
   - Transport Layer
   - Application Layer

4. **Physical Layer Concepts**
   - Transmission Media (Wired and Wireless)
   - Bandwidth, Latency, and Throughput
   - Signal Encoding

5. **Data Link Layer**
   - MAC Addressing
   - Switching
   - VLAN
   - Spanning Tree Protocol

6. **Network Layer**
   - IP Addressing (IPv4 and IPv6)
   - Routing Concepts
   - OSPF, BGP
   - ICMP

7. **Transport Layer**
   - TCP (Transmission Control Protocol)
   - UDP (User Datagram Protocol)
   - Port Concepts

8. **Application Layer Protocols**
   - DNS
   - HTTP/HTTPS
   - FTP
   - SMTP, POP3, IMAP
   - SSH

9. **Network Security**
   - Firewalls
   - VPN
   - Encryption Basics
   - SSL/TLS

10. **Advanced Topics**
    - Load Balancing
    - CDN
    - IoT Networking
    - Software Defined Networking (SDN)
