# Operating Systems Learning Path

## Topics to Study

1. **Introduction to Operating Systems**
   - What is an Operating System
   - Functions of an OS
   - Types of Operating Systems
   - OS Architecture

2. **Process Management**
   - Process vs Thread
   - Process Life Cycle
   - Process States
   - Process Control Block (PCB)
   - Context Switching

3. **CPU Scheduling**
   - Scheduling Criteria
   - Scheduling Algorithms (FCFS, SJF, SRTF, Round Robin, Priority Scheduling)
   - Multi-level Queue Scheduling
   - Thread Scheduling

4. **Process Synchronization**
   - Race Condition
   - Critical Section
   - Mutex and Semaphores
   - Monitors
   - Deadlock (Detection, Prevention, Avoidance)

5. **Memory Management**
   - Physical and Virtual Memory
   - Memory Allocation Strategies (Contiguous, Non-contiguous)
   - Paging
   - Segmentation
   - Page Replacement Algorithms (FIFO, LRU, Optimal)
   - Thrashing

6. **Virtual Memory**
   - Virtual Address Space
   - Page Faults and Demand Paging
   - Memory Protection
   - Swapping

7. **File System**
   - File Concepts
   - File Organization
   - Directory Structure
   - File Allocation Methods (Contiguous, Linked, Indexed)
   - Free Space Management
   - Disk Scheduling (FCFS, SSTF, SCAN, C-SCAN)

8. **I/O Management**
   - I/O Hardware
   - I/O Software Layers
   - Buffering
   - Disk I/O Scheduling

9. **Inter-Process Communication (IPC)**
   - Shared Memory
   - Message Passing
   - Pipes and Named Pipes
   - Sockets

10. **Security and Protection**
    - Access Control
    - Authentication and Authorization
    - Encryption
    - Virus and Malware Protection

11. **Advanced Topics**
    - Distributed Operating Systems
    - Real-time Operating Systems
    - Mobile Operating Systems
    - Cloud OS Concepts
